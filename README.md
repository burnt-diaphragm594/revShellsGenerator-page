# 🔥 Rev Shell's Generator

Generador de comandos de reverse shell moderno, rápido y visual, diseñado para facilitar la creación de payloads en múltiples lenguajes y entornos.

## 🚀 Demo
Disponible vía GitHub Pages.

---

## 🧠 Descripción

**Rev Shell's Generator** es una herramienta web que permite generar automáticamente comandos de reverse shell listos para usar, sustituyendo dinámicamente la IP y el puerto introducidos por el usuario.

Incluye una amplia colección de payloads organizados por categorías como:

- 🐧 Linux / Unix shells
- 🧩 Lenguajes de scripting (Python, PHP, Ruby, Perl, Node.js…)
- 🪟 Windows (PowerShell, Netcat, etc.)
- ⚙️ Lenguajes compilados y herramientas (C, Go, Java, MSFVenom…)

Todo presentado en una interfaz limpia tipo terminal con resaltado de sintaxis y opción de copiado instantáneo.

---

## ✨ Características

- ⚡ Generación instantánea de payloads
- 🌐 Compatible con múltiples lenguajes y sistemas
- 🎯 Sustitución automática de IP y puerto
- 📋 Copiar al portapapeles con un clic
- 🎨 Interfaz estilo terminal moderna
- 🌈 Syntax highlighting avanzado con Highlight.js
- 🧩 Gran variedad de técnicas (TCP, UDP, SSL, FIFO, etc.)
- 🧠 Organización por categorías para fácil navegación

---

## 🖥️ Uso

1. Introduce tu IP
2. Introduce el puerto
3. Selecciona el tipo de payload
4. Haz clic en **Generar** o **Generar Todos**
5. Copia el comando con un clic
