## 🖥️ What This App Does

revShellsGenerator-page is a web app that helps you build reverse shell payloads in more than one language. It replaces the IP and port you enter, so you can create a payload fast. It uses a terminal-style interface and lets you copy the final payload with one click.

This tool fits users who want a simple way to create payload strings for lab work, pentesting practice, or security testing in a controlled setup.

## 📌 How to Use the Fields

### IP Address
Enter the address that your test target should call back to. Use a local address for lab use or a test host address in your setup.

### Port
Enter the listening port number. Pick a port that is free on your machine.

### Language
Select the language or shell type you want. The app changes the payload format for you.

### Copy Button
Use the copy button to place the payload on your clipboard. Then paste it into your test workflow.
